Pokémon Team Planner

FAQ

What is this?

This is a fork of the pokemon team planner made by richi3f that allows for planning cobblemon teams. I removed all original pokemon games, as those can be accessed via his planner.

Will this be updated?

This is a personal project and will most likely not be updated if cobblemon is updates.

If you find the planner useful, please consider making a donation to richi3f.
